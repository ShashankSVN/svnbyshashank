Java Persistence API:
The Java Persistence API provides a POJO persistence model for object-relational mapping. The Java Persistence API was developed by the EJB 3.0 software expert group, but its use is not limited to EJB software components. 
It can also be used directly by web applications and application clients, and even outside the Java EE platform, for example, in Java SE applications.
